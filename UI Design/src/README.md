# 理发店预约App

一个采用Airbnb设计风格的现代化理发店预约应用，提供完整的预约流程和用户体验。

## ✨ 功能特性

### 用户认证
- ✅ 账号密码注册
- ✅ 密码登录
- ✅ 手机验证码登录
- ✅ 密码强度检测
- ✅ 记住登录状态

### 核心预约流程
1. **浏览理发店** - 查看附近理发店列表，支持搜索和排序
2. **查看店铺详情** - 详细的店铺信息、服务项目、理发师团队
3. **选择服务** - 多种服务选择，带详情说明
4. **选择理发师** - 可选择指定理发师或系统自动分配
5. **选择时间** - 日历+时间段选择，实时显示可用性
6. **确认订单** - 汇总预约信息，支持添加备注
7. **预约成功** - 生成预约码，显示完整预约详情

### 预约管理
- ✅ 查看我的预约（待服务/已完成）
- ✅ 预约详情查看
- ✅ 取消预约功能
- ✅ 预约码复制分享

## 📱 页面结构

```
/
├── 登录页 (Login)
├── 注册页 (Register)
├── 店铺列表 (ShopList)
├── 店铺详情 (ShopDetail)
├── 服务选择 (ServiceSelection)
├── 理发师选择 (StylistSelection)
├── 时间选择 (TimeSelection)
├── 确认订单 (OrderConfirmation)
├── 预约成功 (BookingSuccess)
└── 我的预约 (MyAppointments)
```

## 🚀 快速开始

1. **登录/注册**
   - 使用任意邮箱和密码即可注册
   - 支持密码登录或手机验证码登录

2. **开始预约**
   - 从店铺列表选择喜欢的理发店
   - 按照流程选择服务、理发师、时间
   - 确认并提交预约

3. **管理预约**
   - 点击"我的预约"查看所有预约
   - 查看预约详情和预约码
   - 距离预约时间2小时以上可取消

## 📝 数据说明

## 💡 使用提示

1. **时间选择**：今天的已过时间会自动置灰
2. **取消预约**：距预约时间不足2小时无法取消
3. **验证码**：点击"获取验证码"后会有60秒倒计时
4. **预约码**：可以点击复制，用于到店核销
